# homebrew-gdeck

Homebrew tap that distributes [gdeck](https://github.com/Nunokawa-Lab/gdeck) as a **Cask** (pre-built macOS binary).

## Who is this repository for?

Most users install gdeck from the **[gdeck README](https://github.com/Nunokawa-Lab/gdeck#installation)** and never need to open this repository. You are in the right place if you:

- followed a link from the main gdeck project
- ran `brew tap Nunokawa-Lab/gdeck` and want to confirm what this tap provides
- are browsing GitHub and want the install commands for macOS

Usage docs (CLI, TUI, env files, etc.) live in the **[gdeck repository](https://github.com/Nunokawa-Lab/gdeck#readme)** — not here.

## Installation (macOS)

Requires [Homebrew](https://brew.sh/). Go is **not** required; the Cask installs a release binary.

```bash
brew tap Nunokawa-Lab/gdeck
brew trust Nunokawa-Lab/gdeck   # Homebrew 6.0+
brew install --cask gdeck
```

Linux and Windows: see the [gdeck README](https://github.com/Nunokawa-Lab/gdeck#installation) (`go install` or binary download).

## Upgrade

```bash
brew update
brew upgrade --cask gdeck
```

## Uninstall

```bash
brew uninstall --cask gdeck
brew untap Nunokawa-Lab/gdeck
```

## About this tap

| Item | Detail |
